# 悩み系記事テーマ一覧

## 戦略

大家・オーナーの実務悩みキーワードで流入させ、以下の導線でRoomlyに繋ぐ。

```
大家/オーナーが困って検索
  → 対応方法を解説（記事の価値）
    → 「自主管理でこれ全部やるのは大変。管理会社に委託する選択肢もある」
      → 「管理会社はRoomlyでこの業務を効率化できる」

管理会社スタッフが困って検索（副次的リーチ）
  → 同じ記事が実務ガイドとして機能
    → 「Roomlyならこの業務をこう管理できる」
```

### 自主管理大家の実務・ストレス系

自主管理率はユニット数ベースで約48%（815万戸）。家主ベースでも約20%が完全自主管理。小規模個人大家が大半で、修繕対応の遅延（4割超が不満）、24時間対応の精神的負担、法律知識不足が主な痛点。「自主管理 きつい」「大家 ストレス」系のキーワードは競合が個人ブログ中心で狙いやすい。

### 管理委託 × 投資拡大系

自主管理の大家は物件が増えるほど対応負荷が比例して増える。特にクレーム対応のストレスが「次の物件を買う気力」を奪う。2〜3戸で止まってしまう小規模大家は多い。管理会社に委託することで精神的負担から解放され、物件拡大に意識を向けられるようになる——この「管理委託 = コストではなく投資拡大のためのレバレッジ」という切り口は、大家・管理会社の双方に刺さる。

| # | テーマ | 想定キーワード | 狙い |
|---|--------|--------------|------|
| 24 | クレーム対応に疲れて「次の物件」を買えなくなる問題 | 不動産投資 ストレス / 大家 物件 増やせない / 賃貸経営 疲れた | 核心記事。クレーム対応の精神的負担→投資判断の鈍化→管理委託で「経営」に集中→結果的に物件拡大が進む。大家の「管理費5%がもったいない」を「5%で次の物件を買う余裕が生まれる」に転換 |
| 25 | 自主管理から管理委託に切り替えた大家が「最初からこうすればよかった」と思う理由 | 自主管理 管理会社 切り替え / 管理委託 メリット | 体験ベース記事。切り替え前後の生活・精神状態の変化を描写。「空いた時間で物件探し・融資交渉ができるようになった」→管理会社選びのポイント→Roomly |
| 26 | 物件5戸の壁——自主管理で回せる限界と、その先の戦略 | 不動産投資 5戸 / 賃貸経営 規模拡大 / 大家 管理 限界 | 規模別の管理負荷を具体的に試算。1〜3戸は自主管理可→5戸超えると対応漏れ・滞納見落とし→管理委託+ツール化で10戸・20戸への道。投資家コミュニティにも刺さるテーマ |

## 優先順位の考え方

1. **検索ボリューム × 競合の弱さ**で選ぶ
2. 最初は苦情・トラブル系（1, 3, 5）と滞納系（6, 7）から。ボリュームが大きく、管理会社スタッフにも直接刺さる
3. 管理委託判断系（12, 13）はCVに近いが検索ボリュームは小さいので、悩み系記事からの内部リンク先として後から書く
4. 切り口系記事（all-inclusive-service等）は悩み系が揃った後にリライトして内部リンクで繋ぐ
