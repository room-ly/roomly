# 記事テーマ一覧

テーマと切り口案のストック。記事化したいものに `- [x]` をつける → まとめてClaude に渡すと一気にdraftを生成する。

**使い方:**
1. 記事化したいテーマの先頭チェックボックスを `- [x]` に変える
2. 「チェックしたやつdraft化して」とClaude に伝える
3. Claude がチェック済みのテーマを全てdraft化 → draft作成後は削除

切り口は「具体（入口）→ 抽象（本質）」の順で設計する。具体と抽象はなるべく遠いところを行き来する。比喩・比較もなるべく遠い業界・領域から持ち込む。**タイトル自体に「なぜその概念が？」という引っかかりを持たせる。**

## 構造・逆説系（賃貸管理に限定しない）

- [x] ### 隣人の騒音は45デシベルで「うるさい」が、自分の足音は65デシベルでも気にならない
> 同じ音量でも「誰が出したか」で不快度が変わる——騒音問題の本質は音ではない
- 心理音響学の研究：騒音の不快度を決める最大の変数は音量ではなく「コントロール可能性」。自分で出した音は脳が予測して減衰させる（他業界：神経科学・予測符号化）
- 拷問研究における「予測不能な刺激」：同じ電気ショックでも、いつ来るか分からない方が苦痛が大きい（他業界：ストレス心理学）
- 航空機騒音訴訟：同じ騒音レベルでも、空港建設前から住んでいた住民と後から来た住民で訴訟率が違う。「自分が選んだかどうか」が閾値を変える（他業界：環境法）
- **具体（入口）**: 上階の足音に悩む入居者。騒音計で測ると40dB——図書館と同じ静けさ。だが本人は「眠れない」と言う
- **抽象（本質）**: 騒音問題は「音の問題」ではなく「関係性の問題」。不快の正体はデシベルではなく「自分ではどうにもできない」という無力感

- [ ] ### 鍵を3つ付けた家が、最も泥棒に狙われる
> 防御を見せることが「守る価値がある」というシグナルになる逆説
- 犯罪経済学：泥棒は「入れるかどうか」ではなく「リターンが見合うか」で判断する。過剰な防犯装置は「中に高価なものがある」と宣伝している（他業界：ゲーム理論・シグナリング）
- 冷戦の軍拡競争：防衛を強化するほど相手も強化する。セキュリティ・ジレンマ（他業界：国際政治学）
- 生態学の警告色：毒を持つ生物は目立つ色で「食べるな」と伝える。だが無毒の生物が同じ色を真似るベイツ型擬態もある。シグナルは偽装できる（他業界：進化生物学）
- **具体（入口）**: 防犯カメラ4台、ダブルロック、センサーライト完備の一戸建てが空き巣に入られた。隣の無施錠に近い家は無事だった
- **抽象（本質）**: 「守っています」と見せることと「守られている」ことは別の行為。防御のシグナルが攻撃のインセンティブになりうる構造

- [x] ### なぜ「駅徒歩5分」の物件より「駅徒歩12分」の物件の方が退去率が低いのか
> 利便性と満足度が比例しない——「期待値」が居住満足度を決める構造
- 期待不一致理論：満足度は「実際の品質」ではなく「期待との差分」で決まる（他業界：マーケティング・顧客満足度研究）
- ミシュラン効果の研究：星付きレストランは期待が高すぎるため、同じ料理でも満足度が下がる（他業界：行動経済学）
- ピーク・エンドの法則：経験全体ではなく「最も強い瞬間」と「最後の瞬間」で記憶が決まる。毎日の通勤の「最悪の瞬間」が駅近物件では電車の混雑、駅遠物件では静かな帰り道（他業界：認知心理学）
- **具体（入口）**: 駅徒歩3分・築浅・設備充実のワンルームが2年で3回入居者が入れ替わった。駅徒歩14分・築30年のアパートは同じ入居者が8年住んでいる
- **抽象（本質）**: 駅近物件の入居者は「もっと良い物件があるはず」と常に比較している。駅遠物件の入居者は最初から妥協しているため、小さな良さに気づける。不満は現実ではなく期待が作る

- [x] ### 「クレームゼロ」を目標にした管理会社が、3年で顧客の半分を失った
> 問題を消すことと、問題を見えなくすることの区別がつかなくなる構造
- グッドハートの法則：指標が目標になった瞬間、その指標は指標として機能しなくなる（他業界：経済学・政策評価）
- ソ連の釘工場の寓話：「重量」で評価したら巨大な釘だけ作り、「本数」で評価したら使い物にならない極小の釘だけ作った（他業界：計画経済の失敗）
- 病院の「患者満足度」評価：満足度を上げるために不必要な抗生物質を処方する医師が増えた研究（他業界：医療経済学）
- **具体（入口）**: クレーム件数を評価指標にした管理会社。スタッフが入居者からの電話を「相談」として記録し、クレーム台帳に載せなくなった。数字上はクレームゼロ、現場は崩壊
- **抽象（本質）**: 計測できるものを改善しようとすると、計測の仕方を改善してしまう。KPIは現実を映す鏡だが、鏡を磨いても部屋は綺麗にならない

- [x] ### 退去の本当の理由は、退去者に聞いても分からない
> 人は自分の意思決定の理由を正しく説明できない
- 選択盲の実験：被験者に2枚の顔写真から好みを選ばせ、こっそりすり替えて渡す。70%が気づかず、すり替えられた写真について「この目が好き」と理由を語った（他業界：認知心理学・ヨハンソンの実験）
- 退職理由の研究：退職面談で語られる理由と、匿名アンケートで語られる理由が一致しない。面と向かって本音は言えない（他業界：組織心理学・HR）
- 離婚理由の研究：当事者が挙げる理由は「性格の不一致」が最多だが、外部研究者の追跡では経済的要因と距離的要因が支配的（他業界：家族社会学）
- **具体（入口）**: 退去アンケートで「転勤のため」と書いた入居者。実際は隣人トラブルで、同じ市内の別の物件に引っ越していた
- **抽象（本質）**: 人間は自分の行動の理由を「知っている」のではなく「作っている」。退去理由を聞くことは、相手の意思決定を理解することではなく、相手が作った物語を聞くこと

- [ ] ### 家賃を「払わない人」は存在しない——「払えるタイミングが合わない人」がいるだけ
> 滞納を「悪意」で捉えると、回収できたはずの家賃を失う
- 行動経済学のナッジ理論：英国の納税督促状に「あなたの地域の90%は期限内に納税しています」と書いただけで納付率が15%上がった。罰則より社会規範の方が行動を変える（他業界：公共政策・BIT）
- 医療の服薬アドヒアランス：薬を飲まない患者を「不真面目」と見なす医師と、「飲みにくい仕組み」を見直す医師。後者の方が服薬率が高い（他業界：医療行動学）
- 貧困研究：「意志力」は有限のリソース。経済的ストレス下では認知機能がIQ13ポイント分低下する（他業界：行動経済学・セシル＆シャフィール）
- **具体（入口）**: 3ヶ月滞納の入居者に「給料日の翌日に引き落としに変更しませんか」と提案しただけで、翌月から支払いが再開した
- **抽象（本質）**: 滞納を「人の問題」と見るか「仕組みの問題」と見るかで、回収率が変わる。道徳で裁くと解決策が「追い出す」しか残らないが、設計で見ると選択肢が広がる

- [ ] ### 空室を「埋める」と言っている限り、空室は減らない
> 言語が思考を規定する——賃貸経営における言葉の罠
- サピア＝ウォーフ仮説：使う言語が思考の枠組みを決める（他業界：言語学・認知科学）
- 軍事用語の設計：「撤退」を「転進」と呼び変えた日本軍。言葉を変えると現実認識が歪む（他業界：軍事史）
- 医療の「患者」vs「利用者」：呼び方を変えたら医療者の態度が変わった研究（他業界：医療社会学）
- **具体（入口）**: 「空室を埋めたい」と相談に来たオーナーに「埋めるのではなく、選ばれる部屋にするのでは？」と言い換えただけで、リフォームの方向性が変わった
- **抽象（本質）**: 「埋める」は穴を塞ぐイメージ——受動的で、誰でもいいから入れるニュアンスを含む。言葉が変わると問いが変わり、問いが変わると解が変わる
